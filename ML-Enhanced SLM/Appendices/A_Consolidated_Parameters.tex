\begin{keyinsight}
    Several patterns emerge from the consolidated tables:
    \begin{itemize}
        \item All DNN-based approaches use the \textbf{Adam optimizer}
              with learning rate $10^{-3}$---this has become the de facto
              standard in the field.
        \item Subcarrier counts remain moderate ($N \le 256$) across all
              studies, leaving scalability to practical system sizes
              ($N = 2048$+) as an open question.
        \item PR-DUN stands out with dramatically fewer trainable parameters
              ($\sim 20$) compared to all other neural approaches
              ($10^5$--$10^6$).
    \end{itemize}
\end{keyinsight}
